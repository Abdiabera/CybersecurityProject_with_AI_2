Lab Exercises 3 — How Devices Communicate Across Layers

Module 5 — Practical Activity Cisco Packet Tracer 2 Hands-On Tasks
From Dr. Solomon Seifu — These two tasks build directly on top of each other. Task 1 shows you how two devices talk on the same network. Task 2 adds a router so you can see what changes when traffic must cross between networks. Pay close attention to what happens at Layer 2 vs. Layer 3 — that is the core concept of this module.
