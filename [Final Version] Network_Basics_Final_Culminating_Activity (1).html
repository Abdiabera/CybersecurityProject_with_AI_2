Exercise 2 — Packet Flow Sequencing

Put the Steps of a TCP Connection in the Correct Order

A TCP connection involves a specific 3-step process before data can be sent. Number these events in the correct order by thinking through the TCP three-way handshake. Then click Reveal to verify.

☐  A. The client sends a SYN packet to the server to initiate a connection
☐  B. The server responds with a SYN-ACK packet acknowledging the request
☐  C. The client sends an ACK packet and the connection is established
☐  D. Data transmission begins between client and server
✅ Correct Order: A → B → C → D

Step 1 (A): The client sends a SYN (synchronize) packet — "I want to connect."
Step 2 (B): The server responds with SYN-ACK — "I hear you, I'm ready."
Step 3 (C): The client sends ACK — "Acknowledged, let's go." Connection established.
Step 4 (D): Data now flows reliably in both directions over the established connection.
Exercise 3 — Identify the Device

Read Each Description and Identify the Correct Network Device

Each clue describes one network device. Name the device and then reveal the answer to verify.

Clue 1: I operate at Layer 2. I learn which MAC address is on which port and only forward frames to the correct destination, reducing unnecessary traffic on the LAN.
✅ Answer: Network Switch
Clue 2: I operate at Layer 3. I forward packets between different networks and make decisions based on IP addresses, not MAC addresses.
✅ Answer: Router
Clue 3: I have no intelligence — I receive a signal on one port and broadcast it out of every other port. I operate at Layer 1 only.
✅ Answer: Hub
Clue 4: I connect your home network to your ISP. I translate signals between your home's network and the ISP's infrastructure (cable or DSL).
✅ Answer: Modem
Clue 5: I allow wireless devices to connect to a wired network. I am commonly found in offices and classrooms and extend network access without a physical cable.
✅ Answer: Wireless Access Point (WAP)

How to Facilitate This Activity

Suggested Timing: Allocate 90–120 minutes for the full portal. Recommended breakdown: Part 1 (25 min) → Part 2 (20 min) → Part 3 (15 min) → Part 4 (20 min) → Part 5 Reflection (15 min) → Full class debrief (10 min).
  • After Part 1, pause and ask the class: "Which question surprised you most, and why?"
  • For Part 2 scenarios, encourage learners to explain their reasoning out loud — this is where real understanding is revealed.
  • Part 3 Thinking Exercises work well as pair activities. Let learners discuss before revealing answers.
  • Use the Challenge Round as a competition element — first learner to get all 8 correct wins recognition.
  • The Reflection section is private. Encourage honest self-assessment. Use the checked "Topics to Review" as your guide for the final review session.
  • After all parts, hold a 10-minute debrief asking: "Name one concept that fully clicked for you today. Name one concept you still want to revisit."
